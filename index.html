꼬들이란?

꼬들은 여섯 번의 기회 안에 오늘의 두 글자 한글 단어를 맞히는 무료 웹 퍼즐 게임입니다. 흔히 한글 워들, 한국어 워들이라고 부르는 단어 맞히기 게임으로, 영어 워들(Wordle)의 한글판이라고 생각하면 쉽습니다. 단어는 자음·모음 6개 낱자로 이루어져 있고, 낱자를 입력하면 위치까지 정확한 낱자는 초록, 단어에 있지만 위치가 다른 낱자는 노랑, 단어에 없는 낱자는 회색으로 표시됩니다. 매일 자정(한국 시간)에 새 문제가 공개되며, 전 세계 모든 플레이어가 같은 오늘의 문제를 풉니다. 회원가입과 설치 없이 브라우저에서 바로 즐기는 데일리 낱말 퍼즐이라 출퇴근이나 잠깐의 휴식 시간에 부담 없이 한 판 하기 좋습니다.

어떻게 플레이하나요?

  1. 화면 키보드나 실제 키보드(두벌식)로 낱자 6개를 입력합니다. 예: 강물 = ㄱ·ㅏ·ㅇ·ㅁ·ㅜ·ㄹ
  2. 복합모음은 나누어 입력합니다. 과일 = ㄱ·ㅗ·ㅏ·ㅇ·ㅣ·ㄹ (ㅘ → ㅗ+ㅏ)
  3. 겹받침도 나누어 입력합니다. 여덟 = ㅇ·ㅕ·ㄷ·ㅓ·ㄹ·ㅂ (ㄼ → ㄹ+ㅂ)
  4. 쌍자음(ㄲ ㄸ ㅃ ㅆ ㅉ)은 한 낱자이며 ⇧ 키로 입력합니다.
  5. 색깔 힌트를 활용해 여섯 번 안에 단어를 완성하세요.

자주 묻는 질문

한글 워들이 뭔가요?
한글 워들은 영어 단어 맞히기 게임 워들을 한국어로 옮긴 게임입니다. 꼬들은 알파벳 대신 한글 자음·모음을 맞히는 한국어 워들로, 두 글자 단어를 자모 6낱자로 풀어 여섯 번 안에 알아맞힙니다.
꼬들 하는 법이 궁금해요.
화면 키보드나 두벌식 키보드로 낱자 6개를 입력해 단어를 추측하고, 초록·노랑·회색 색깔 힌트를 보고 여섯 번 안에 오늘의 단어를 맞히면 됩니다. 아래 어떻게 플레이하나요? 단계별 설명도 참고하세요.
새 문제는 언제 나오나요?
매일 자정(한국 표준시)에 새 문제가 나옵니다. 하루 한 문제이며, 전 세계 모두가 같은 오늘의 문제를 풉니다.
오늘의 꼬들 정답은 어디서 보나요?
오늘의 정답은 문제를 풀거나 여섯 번의 시도를 모두 쓰면 바로 공개됩니다. 어제까지의 지난 정답은 지난 문제 페이지에서 회차별로 모두 확인할 수 있습니다.
진행 기록은 어디에 저장되나요?
플레이 기록·연속 정답·정답 분포 통계가 브라우저에 자동 저장됩니다. 서버로 전송되지 않습니다.
색약이 있는데 색 구분이 어려워요.
통계 창에서 고대비 모드를 켜면 초록/노랑 대신 주황/파랑으로 표시됩니다.
